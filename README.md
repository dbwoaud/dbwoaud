### 🚀 Core Strengths (Technical Points)

* **UI Automation**: 리플렉션과 네이밍 컨벤션을 활용한 UI 자동 바인딩 시스템 구축으로 작업 효율 극대화.
* **Architecture**: SOLID 원칙을 준수한 매니저 패턴 설계 및 싱글톤 베이스 클래스 활용.
* **Event-based System**: C# Action과 ScriptableObject를 결합하여 시스템 간 의존성을 낮춘(Decoupled) 데이터 주도 설계.
* **Optimization**: 자료구조(Stack) 기반의 반복적 DFS 알고리즘을 직접 구현하여 씬 전체 탐색 비용을 $O(M)$ 수준으로 최적화.
